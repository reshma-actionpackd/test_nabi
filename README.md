# Basic Web Calculator

A simple calculator built with vanilla HTML, CSS, and JavaScript. No frameworks or build step required.

## Features

- Basic operations: add, subtract, multiply, divide
- Clear, sign toggle, percent
- Keyboard support
- Dark, phone-style UI

## Run locally

### Option 1: Open in browser

Double-click `index.html` or open it in any modern browser.

### Option 2: Dev server (optional)

```bash
npm install
npm start
```

Then open [http://localhost:3000](http://localhost:3000).

## Project structure

| File | Description |
|------|-------------|
| `index.html` | Calculator markup |
| `styles.css` | Layout and theme |
| `calculator.js` | Calculator logic |
| `package.json` | Optional `serve` dev server |

## Keyboard shortcuts

| Key | Action |
|-----|--------|
| `0`–`9`, `.` | Digits / decimal |
| `+`, `-`, `*`, `/` | Operators |
| `Enter` or `=` | Equals |
| `Escape` | Clear |
| `Backspace` | Delete last digit |
